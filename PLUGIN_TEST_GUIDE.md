## 🐛 故障排查

### 问题 1：插件登录后立即退出

**原因：** Token 存储问题

**解决：**
1. 打开 Chrome DevTools (F12)
2. 切换到 Console 标签
3. 查看是否有错误信息
4. 尝试清除插件数据：
   - `chrome://extensions/`
   - 点击 AiMaster 的"详细信息"
   - 点击"清除存储空间"

### 问题 2：爬取失败

**原因：** 平台检测失败或爬虫错误

**解决：**
1. 确认 URL 是否匹配平台规则
2. 打开 DevTools 查看错误
3. 手动测试爬虫：
   ```javascript
   // 在文章页面的 Console 中运行
   const result = window.NetEaseCrawler?.crawl();
   console.log(result);
   ```

### 问题 3：后端连接失败

**原因：** 后端服务未启动或 CORS 问题

**解决：**
1. 确认后端运行中：`curl http://localhost:8000/`
2. 检查 API 地址是否正确
3. 查看 DevTools Network 标签，确认请求是否发送

### 问题 4：任务一直处理中

**原因：** 后端任务处理异常

**解决：**
1. 查看后端日志
2. 手动测试任务 API：
   ```bash
   # 获取任务详情
   curl http://localhost:8000/api/tasks/{task_id}
   ```
